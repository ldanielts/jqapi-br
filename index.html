jQAPI-BR - Documentação alternativa do jQuery - Versão 1.6

Por que outra documentação?Resolver seus problemas é muito divertido.

Existem várias documentações alternativas por ai, como a jQuery API Browser, jQuery API, Visual jQuery entre outras como jQuery API Browser. Elas estão desatualizadas. A última documentação oficial foi lançada no 14 days of jQuery. Por que outra?

A documentação do jQuery é boa, muito completa, com muitos exemplos e demontrações. A única coisa que me entriga é a maneira que devemos procurar pela documentação certa para o que estamos procurando. Tente procurar pela função .is(). Mais de 100 resultado antes da função desejada?! *) Onde tenho que rolar página abaixo procurando pelo que preciso. Tem de existir uma maneira melhor, obviamente.

A documentação alternativa do Rails, a Rails API é minha favorita para minhas ferramentas server-side. Infelizmente minhas ferramentas client-side não tem uma documentação rápida e bonita. Inspirado pela Rails API e desesperado por uma navegação melhor eu codifiquei a minha. Claro que com jQuery!

Só para deixar claro, eu apenas codifiquei a navegação e reestilizei a documentação. O conteúdo é o mesmo da documentação oficial e todos os créditos pertencem ao jQuery team. Continuem com o belo trabalho!

*) Edit: isso é o primeiro resultado quando pesquisado entre aspas.

Uso do tecladoDeixando as coisas rápidas.

As teclas que funcionam caso o campo de busca esteja com foco ou o cursor do mouse esteja na área de navegação. ESC é tecla global. 

CIMA    - Categoria anterior / Item anterior
BAIXO   - Próxima categoria / Próximo item
ENTER   - Abrir categoria / Carregar item selecionado
ESC     - Limpa os resultados da busca e coloca o campo em focus
