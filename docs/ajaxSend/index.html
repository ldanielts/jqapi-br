.ajaxSend()

Categorias: Ajax > Manipuladores de Eventos Globais do Ajax

.ajaxSend( handler(event, jqXHR, ajaxOptions) ) Retorna: jQuery

Descrição: Inclui uma função a ser executada antes que a requisição Ajax seja enviada. Isso é um evento Ajax.

  • versão adicionada: 1.0.ajaxSend( handler(event, jqXHR, ajaxOptions) )

    handler(event, jqXHR, ajaxOptions)A função a ser invocada.

Sempre que uma requisição Ajax está prestes a ser enviada, JQuery dispara o evento ajaxSend . Qualquer e todos os manipuladores que tenham sido registrados com o método.ajaxSend() são executados nesse momento.

Para observar este método em ação, podemos criar uma requisição básica de carga Ajax:

<div class="trigger">Trigger</div>
<div class="result"></div>
<div class="log"></div>

Podemos incluir nosso manipulador de evento a qualquer elemento:

$('.log').ajaxSend(function() {
  $(this).text('Triggered ajaxSend handler.');
});

Agora, nós podemos criar uma requisição Ajax usando qualquer método JQuery:

$('.trigger').click(function() {
  $('.result').load('ajax/test.html');
});

Quando o usuário clica no botão e a requisição Ajax está prestes a iniciar, a mensagem de log é exibida.

Nota: Porque .ajaxSend() é implementado como um método das instâncias do JQuery, nós podemos usar a palavra reservada this como fazemos aqui para nos referir aos objetos selecionados dentro da função de callback.

Todos os manipuladores ajaxSend são invocados, independente de qual requisição Ajax está prestes a ser enviada. Se somos obrigados a diferenciar entre as requisições, nós podemos usar os parâmetros passados ao manipulaor. Cada vez que um manipulador ajaxSend é executado, é passado o objeto do evento, o objeto jqXHR (na versão 1.4, XMLHttpRequest), e o objeto de configuração que foi usado na criação da requisição Ajax. Por examplo, podemos restringir nosso callback para apenas tratar eventos que lidam com uma URL em particular:

$('.log').ajaxSend(function(e, jqxhr, settings) {
  if (settings.url == 'ajax/test.html') {
    $(this).text('Triggered ajaxSend handler.');
  }
});
