.fadeIn()

Categorias: Efeitos > Fading (desvanecimento)

.fadeIn( [ duration ], [ callback ] ) Retorna: jQuery

Descrição: Exibe os elementos alvo de um seletor jQuery os desvanecendo para opaco.

  • criado na versão: 1.0.fadeIn( [ duration ], [ callback ] )

    durationUma string ou número determinando por quanto tempo a animação vai rodar.

    callbackUma função a ser chamada logo que a animação terminar.

  • criado na versão: 1.4.3.fadeIn( [ duration ], [ easing ], [ callback ] )

    durationUma string ou número determinando por quanto tempo a animação vai rodar.

    easingUma string indicando qual função de atenuação usar para a transição.

    callbackUma função a ser chamada logo que a animação terminar.

O método .fadeIn() anima a opacidade dos elementos alvo de um seletor JQuery.

As durações são dadas em milissegundos; valores mais altos indicam animações mais demoradas, não as mais rápidas. As strings strings 'fast' e 'slow' podem ser fornecidas para indicar durações de 200 e 600 milissegundos, respectivamente. Se qualquer outra string for informada, ou se o parâmetro duration for omitido, a duração default de 400 milissegundos será usada.

Desde a versão 1.4.3 do JQuery, uma string opcional contendo o nome de uma função de atenuação pode ser usada. Funções de atenuação especificam a velocidade na qual a animação progride em diferentes pontos na animação. As únicas implementações de atenuação na biblioteca jQuery são as default, chamadas swing, e uma que progride num ritmo constante, chamada linear. Mais funções de atenuação estão disponíveis através do uso de plug-ins, mais notavelmente a jQuery UI suite.

Se fornecida, a função callback é disparada logo que a animação termina. Isto pode ser útil para amarrar diferentes animações em sequência. A função callback não recebe argumentos, mas this representa o elemento DOM sendo animado. Se múltiplos elementos forem animados, é importante notar que o callback é executado uma vez para cada elemento, não uma vez para a animação inteira.

Nós podemos animar qualquer elemento, como uma imagem simples:

<div id="clickme">
      Clique aqui
    </div>
    <img id="book" src="book.png" alt="" width="100" height="123" />
    Com o elemento invisível no início, nós podemos mostrá-lo devagar:
    $('#clickme').click(function() {
      $('#book').fadeIn('slow', function() {
        // Animação completa
      });
    });

Notas adicionais:

  • Todos os efeitos jQuery, incluindo .fadeIn(), podem ser desligados globalmente setando jQuery.fx.off = true, o que efetivamente seta a duração para 0. Para mais informações, veja jQuery.fx.off.

Exemplos:

Exemplo: Anima divs ocultas desvanecendo-as uma por uma, completando cada animação em 600 milissegundos. 

<!DOCTYPE html>
<html>
<head>
  <style>
    span { color:red; cursor:pointer; }
    div { margin:3px; width:80px; display:none;
      height:80px; float:left; }
      div#one { background:#f00; }
      div#two { background:#0f0; }
      div#three { background:#00f; }
    </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <span>Click here...</span>

    <div id="one"></div>
    <div id="two"></div>
    <div id="three"></div>
<script>
      $(document.body).click(function () {
        $("div:hidden:first").fadeIn("slow");
      });
    </script>

</body>
</html>

Example: Fades a red block in over the text. Once the animation is done, it quickly fades in more text on top. 

<!DOCTYPE html>
<html>
<head>
  <style>
      p { position:relative; width:400px; height:90px; }
      div { position:absolute; width:400px; height:65px; 
        font-size:36px; text-align:center; 
        color:yellow; background:red;
        padding-top:25px; 
        top:0; left:0; display:none; }
        span { display:none; }
      </style>
  <script src="http://code.jquery.com/jquery-git.js"></script>
</head>
<body>
  <p>
        Let it be known that the party of the first part
        and the party of the second part are henceforth
        and hereto directed to assess the allegations
        for factual correctness... (<a href="#">click!</a>)
        <div><span>CENSORED!</span></div>

      </p>
<script>
        $("a").click(function () {
          $("div").fadeIn(3000, function () {
            $("span").fadeIn(100);
          });
          return false;
        }); 

      </script>

</body>
</html>
